# Minecraft 台球数据包

> 在 **Minecraft Java 版** 中体验接近现实的台球运动，包含自定义球体物理、旋转效果、自动规则判定和电脑玩家。  
> 支持 **斯诺克、英式八球、花色八球、九球** 和自定义练习模式。  
> 支持 **单人游戏、人机对战和双人 PvP 游戏**。

默认数据包通过可点击的触发器命令支持生存模式玩家和非 OP 玩家。Minecraft 1.21 及更高版本还支持可调整的球半径、动态球桌缩放，以及基于物品展示实体改进的视觉效果。

# 安装

> **更新数据包后：** 请完全移除已有球桌并重新放置。这可以确保球桌使用新的实体类型、袋口位置、渲染系统和缩放逻辑。

- Step 01 下载代码压缩包并解压；
- Step 02 打开Minecraft路径，通常是 "C:/Users/youUserName/AppData/Roaming/.minecraft"；
- Step 03 将资源包文件夹 `Pool-ResourcepackFolder-Squid-Workshop-（适配的游戏版本）` 放入 **.../resourcepacks** 文件夹；
- Step 04 选择一个世界安装数据包，进入 **.../saves/world/datapacks** 文件夹；
- Step 05 将数据包文件夹 `Pool-Datapack-Squid-Workshop-（适配的游戏版本）` folder 放入 **.../saves/world/datapacks** 文件夹，每个世界都需要一个独立的数据包副本；
- Step 06 打开Minecraft，进入对应世界；
- Step 07 按下Esc，进入设置.../**资源包...**，将资源包压缩文件移动到界面右上角，每次更换游戏版本都需要重新设置；
- Step 08 使用 `/reload` 指令加载数据包并根据安装提示进行设置; 
- Step 09 开始游玩。

如果不能正常运行，使用 `/datapack list` 确认数据包是否开启，按需使用 `/datapack enable "datapackname"` 开启。 \
可以选择将数据包和资源包压缩成zip文件，显著减少文件大小。

### 兼容性注意
- **通用说明：** 本数据包和资源包在多个支持的游戏版本中均可正常运行，但 `pack.mcmeta` 文件可能未及时更新。若未来 Minecraft 版本仅需修改 `pack.mcmeta` 即可兼容，可能不会专门为此发布新版本。因此，在安装时游戏可能会提示资源包或数据包“不兼容”。如需消除该提示，可参考[此页面](https://minecraft.fandom.com/wiki/Pack_format)，根据当前游戏版本手动修改 `pack.mcmeta` 文件。
- **1.21:** 现已提供整合版本的数据包 `Pool-Datapack-Squid-Workshop-1.21plus` 和资源包 `Pool-ResourcepackFolder-Squid-Workshop-1.21plus`，可以在1.21的所有版本使用并且兼容后续版本。`Releases_1.21` 中提供的的子版本已被移除。

**如果数据包或资源包与Java Edition 1.16以上的任意版本不兼容，请告知我。**

### 8. 开始人机对战

数据包支持在 **八球和九球** 游戏中与电脑玩家对战。

1. 放置一张球桌并打开游戏大厅。
2. 选择八球或九球单人游戏。
3. 选择 **人机对战**。

### 10. 击球
<video src="Gallery/Videos/user_guides_CN/10_shoot_ball_CN.mp4" alt="_MingshiYangUIUC_" width="75%" autoplay loop muted playsinline></video>


https://github.com/user-attachments/assets/456d5460-56d5-48b4-8102-26a629dada0c


使用鼠标右键蓄力，松开右键出杆。击球时母球的初始运行方向由玩家朝向角度决定，母球的初始速度由按住右键的时间决定，最大力度的击球需要1.1秒左右的蓄力。八球和九球开球时速度会提升一个可以调整的比率。  
瞄准方法：手持球杆按住潜行键时，玩家会朝向母球，此时击球时母球运动方向即为正前方。

您可以设置是否在下蹲时添加缓慢效果来放大屏幕，这可以帮助您更精准的瞄准，但是在网络延迟较大时不建议使用。通过补全以下指令完成设置：

	/function app:settings/pool/aim/

### 14. 提示
- 一些实用指令:
	1. 直接打开命令窗口，不使用聊天栏的按钮   
	`/function app:help/pool/commandwindow`
	2. 手动运行加载函数（用于排查问题 / 检查缺失设置，而无需执行`reload`）   
	`/function pool:classes/main/load` 或者点击命令窗口中的<<ins>(重新)加载台球数据包</ins>>    
	3. 在不使用命令窗口的情况下结束当前对局   
	`/function pool:classes/lobby/terminate_helper`
	4. 在不使用命令窗口的情况下移除球桌   
	`/function pool:classes/table/clear`

- 在开始使用数据包前，每个世界都需要设置游戏版本（仅适用于 mc1.16–1.20）以及摩擦参数。每位进入该世界的玩家也需要分别完成自己的语言设置。
- 在游戏中，准星（**+**）并不完全等同于玩家在世界坐标中的真实朝向。例如在 1.21.11 版本中，朝向略微偏移（对齐竖线的右侧）。建议适当调整瞄准方向，或在设置中手动调整母球偏移来进行校正，建议值基于16:9全屏游玩的场景。
- 推荐使用 Minecraft 1.21 及以上版本进行游玩，因为基于 item display 实现的画面效果会有明显提升。本数据包1.20.5之前的版本不支持配方系统。
- 建议关闭“强制 Unicode 字体”以获得更佳体验。
- 斯诺克游戏开始时，球桌上摆放的红球数量取决于球桌大小。当玩家放置母球时，彩球的原始点位上会出现紫色粒子，提示球桌的布局。
- 更改世界的游戏版本，或更新数据包与资源包，可能会导致旧的对局、球桌或装备出现异常。如果发现显示不正确，请重新设置游戏版本、重建球桌，并重新获取球杆。
- 如果你要在资源包中添加自定义模型或贴图，请注意游戏版本，因为不同版本可能使用不同的模型和贴图目录。支持的版本范围以 `vx_vy` 的形式标注。对于较新的版本，如果未特别指定，将使用默认目录。
- 如果不确定自由球的可放置区域，开球区（kitchen）和/或 D 区位于球桌的正 X 或正 Z 方向一侧（即开球方向朝向负方向）。
- 如果有多名玩家同时在球桌上放置球，请不要靠得太近，否则视线判定指示器可能会出现异常。

### 15. 非 OP 与生存模式玩家

从 **v1.2.0** 开始，默认数据包通过 trigger 指令支持非 OP 玩家与生存模式玩家。

- 几乎所有功能都可以在没有指令权限的情况下使用，只有少数不常用设置除外。
  - 一些高级设置，例如 teleport_duration 修改或 gamerule 相关选项，可能仍然需要指令权限。
- 建议在创建世界时，通过 **数据包（Data Packs）** 菜单配置数据包。
  - 在一些较旧的 Minecraft 版本中，如果在创建世界后再添加数据包，可能需要运行 `/datapack enable` 和 `/reload`。
  - 这些指令需要更高权限，因此在创建世界时就添加数据包，可以获得更好的非 OP 使用体验。
- 玩家可以在**没有指令权限**的情况下直接打开命令窗口：   
	`/trigger swPool__trigger set 1`
- 如果还没有选择语言，并且命令窗口无法正常显示，数据包会先提示玩家选择语言。
- 如果开启了配方要求，那么在非创造模式下，玩家将无法再通过命令窗口直接获取球杆或设置球桌。
  - 玩家需要通过合成获得球杆和球桌核心。
  - 和工作台互动时可以查看所有配方。
- 如果玩家在生存模式中游玩，请在游戏过程中保持副手栏为空。
  - 数据包可能会向玩家的副手栏给予箭矢物品，用于击球。
  - 这些箭矢看起来像绿色（`√`），并且手持时对其他玩家不可见。

#### 合成球杆

使用木棍和染料可以合成球杆。

<img src="Gallery/Images/stick_recipe.webp" alt="球杆合成配方" width="50%">

球杆是不可破坏的弓。游戏过程中，数据包会提供箭矢让玩家完成击球操作。

#### 合成球桌核心

使用地毯、铁锭和基础材料可以合成球桌核心。

<img src="Gallery/Images/table_recipe.webp" alt="球桌核心合成配方" width="50%">

球桌核心是一个自定义箱子物品。它会根据合成时使用的材料记录球桌的台布和边框信息。

摆放球桌的方法：

1. 将球桌核心放置在玩家想要的位置。
2. 系统会显示设置提示。
3. 根据提示确认摆放球桌。
4. 如果玩家改变主意，请在确认摆放前，从箱子中取出物品。

确认摆放后，箱子和其中的物品会被移除，球桌会在箱子所在位置生成。

注意：临时的球桌摆放辅助实体会持续 60 秒。如果过期，请从箱子中取出球桌核心物品，然后重新放置一次。

#### 限制服务器中的球桌管理权限

服务器管理员可以选择启用白名单限制，用于控制球桌放置、球桌配置和全局设置。

启用后：

- 启用该限制的 OP 会自动加入白名单。
- OP 可以通过 `swPool_whitelisted` 玩家标签授予或撤销权限。
- 白名单玩家可以使用受保护的设置和球桌管理功能。
- 其他玩家仍然可以使用玩家个人设置和普通游戏控制。

白名单的启用和关闭选项需要命令权限，因此普通非 OP 玩家无法操作。

授予权限：`/tag <玩家> add swPool_whitelisted`   

撤销权限：`/tag <玩家> remove swPool_whitelisted`

# 常见问题

1. 我无法击球。 \
请站在地面，在结束蓄力的瞬间请保持站立或者下蹲姿势不变。

2. 球的运动看起来不太正常。 \
请确认你已经成功安装/加载了 **数学数据包/模块**（见上方说明），在执行 `reload` 后你应该能看到来自两个数据包的提示信息。在极少数情况下，如果球桌上同时存在过多的球，继续提高游戏规则中的 `maxcommandchainlength` (新版本：`max_command_sequence_length`) 可能会有所帮助。

3. 我无法撤销一次击球。 \
请在设置菜单中允许 `作弊(cheat)`。在对局过程中撤销击球被视为一种“作弊”行为。

4. 执行 `reload` 之后一切都消失了。 \
请在设置菜单中选择 `keepongoingsession`。

5. 数据包在运行，但效果和视频中不一样。 \
请尝试调整可用的设置，尤其是 `visuals` 相关选项。需要注意的是，不同的物理参数有时也会导致明显不同的效果。

6. 我的球桌坏掉了。 \
请将球桌搭建在平整且足够大的表面上，并且不要使用任何工具对球桌进行“挖掘”。

7. 我需要摆球但是背包里没有球。 \
请使用 `/function app:get/pool/cueball_helper` 获得新的母球。

8. 我遇到了意料之外的问题。 \
请尝试撤销上次击球并再试一次、重置对局、重置球桌，或卸载 / 重新加载数据包。如果问题仍然存在，请提交一个 _issue_ 或直接联系我。更推荐提交 issue，并附上问题的录屏或世界存档，帮助我确认并解决问题。

9. 游戏日志记录了很多报错。
这是正常现象，因为数据包中包含适配不同游戏版本的函数，一些文件在当前版本下可能无法加载，从而产生报错信息。

10. 电脑玩家出杆等待时间很长，或者造成暂时卡顿。  
电脑玩家会通过内部物理模拟来评估击球。可以减少评估的瞄准点数量、使用 5 组合动作空间，或者增加分配给每次模拟的游戏刻数。请避免将摩擦系数设置得过低，因为模拟中的球会需要更长时间才能停止。

11. 电脑玩家思考结束后，游戏突然加速。  
游戏可能正在补回长时间模拟所耽误的游戏刻。请增加电脑玩家设置中的每次模拟刻数，为各次模拟之间提供更多恢复时间。

12. 我无法开始与电脑玩家对战。  
必须启用自动裁判。目前电脑玩家支持八球和九球，并且是围绕默认球半径 `1250` 进行调试的。

13. 球的半径或袋口尺寸与球桌不匹配。  
球半径和球桌缩放设置只会影响新放置的球桌。更改这些设置后，请移除现有球桌并重新放置。

# 注意事项和使用条款

本数据包使用简化的牛顿力学和神经网络进行建模，忽略了袋口摩擦、库边压缩、球与球碰撞时的变形、能量损失等因素，力求在 Minecraft 中实现近似的真实物理行为。

目前已模拟以下机制：
- 动量守恒、角动量守恒与动能传递
- 母球击球点与球杆抬起角度影响对母球旋转的影响
- 与袋口边缘的交互（但不包含袋口摩擦力）

开球的物理模拟参考[此链接](https://math.stackexchange.com/questions/658871/perfectly-centered-break-of-a-perfectly-aligned-pool-ball-rack/659318)。其它物理部分为原创设计，未参考文献。如需深入了解可参考基础力学课本，或搜索关键词如 “台球物理”。

默认球半径经过有意放大，以提高可玩性。虽然更小的球可能更接近现实中的台球尺寸，但在那种尺度下，Minecraft 的键鼠灵敏度会让精确瞄准变得更加困难。在 Minecraft 1.21 及更高版本中，可以调整球的半径。

数据包内各种台球游戏的规则已作简化，以提升可玩性与易用性。您可以参考：[斯诺克规则](https://zh.wikipedia.org/wiki/%E6%96%AF%E8%AF%BA%E5%85%8B)、[英式八球规则](https://zh-yue.wikipedia.org/wiki/%E8%8B%B1%E5%BC%8F16%E6%B3%A2)、[通用八球规则](https://zh.wikipedia.org/wiki/8%E8%99%9F%E7%90%83)、[九球规则](https://zh.wikipedia.org/wiki/9%E8%99%9F%E7%90%83)。

资源包中包含简化后的球桌袋口图示，贴近 2005 年斯诺克标准球桌袋口尺寸。您可以在资源贴图文件中查看。
